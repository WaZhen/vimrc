# Instructions

Copy de .vimrc in ~/
Install vundle: https://github.com/VundleVim/Vundle.vim
Open vim and type the command:
:PluginInstall
If you want to enable code completion follow these instructions:
https://github.com/ycm-core/YouCompleteMe#linux-64-bit
Otherwise modifiy the .vimrc file and comment out the line where the YouCompleteMe plugin is
